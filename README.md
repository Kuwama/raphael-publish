## 🎯 How to Use raphael-publish

Here is a quick guide to use the application:

1. Launch raphael-publish from your desktop or Start menu.

2. You will see a simple window prompting you to open or create a Markdown file.

3. To start, click "New" or "Open" if you have an existing Markdown file.

4. Write your content in plain Markdown text. Use simple formatting like:

   - `#` for headings
   - `*` or `-` for bullet lists
   - `![alt text](image link)` for images
   - `[link text](URL)` for links
   - Triple backticks for code blocks

5. On the right side (or in a preview tab), see your content formatted as it will appear in your article.

6. Adjust styles or layout options if needed using the tools provided.

7. When ready, choose "Export" to save the formatted article in a compatible file type.

8. Copy or upload this exported content to your WeChat public account or other platforms.
